Sentiment Chatbot – Python Project

Features:
• Detects Positive, Negative, Neutral sentiment
• Responds emotionally and professionally
• Lightweight, fast, and suitable for customer support chatbots

How to Run:
1. Install dependencies:
   pip install nltk

2. Run the chatbot:
   python main.py

3. Type messages and see sentiment-aware responses.

Project Files:
• sentiment_analyzer.py – VADER sentiment detection
• model.py – Response logic
• chatbot.py – Connects sentiment + reply generator
• main.py – Chat interface
